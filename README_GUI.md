1. 將 **Wartales 戰爭傳說 重注中文字體包.zip** 解壓縮到遊戲資料夾（`Wartales.exe` 所在位置）。
2. 執行 **`Wartales_repack_font_gui.exe`** 以開啟圖形介面。
3. 選擇你想使用的字體大小與 TTF（字型家族 + 粗細）。
4. 點擊 **「重打字體」** 按鈕以開始重新打包流程。


1. unzip Wartales 戰爭傳說 重注中文字體包.zip into the game folder (where `Wartales.exe` is located).
2. run `Wartales_repack_font_gui.exe` to open the GUI.
3. select the font size and TTF (font family + weight) you want to use.
4. click the "重打字體" button to start the repacking process.
